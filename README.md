# Rellat Otter
Rellat Alpha with WebRTC. Simple Collaborative Code editor.

You can test it on http://ide.rellat.com:3000/

## Install

install node.js and npm.

```
git clone --recursive https://github.com/kifhan/Rellat-otter.git
cd Rellat-otter
npm install
node index.js
```
